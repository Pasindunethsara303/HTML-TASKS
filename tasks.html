Ordered Q&A List (Nested)

  1. What is HTML?
    • HTML stands for HyperText Markup Language.
  2. What is the use of the <a> tag?
    • It defines a hyperlink.
  3. What is a semantic tag?
    • Tags that have meaning, like <article>, <section>.
  4. What does the alt attribute do?
    • It provides alternative text for an image.
  5. How do you merge cells in a table?
    • Use `colspan` or `rowspan` attributes.
  6. What is the difference between ordered and unordered lists?
    • Ordered lists are numbered; unordered lists use bullets.
  7. How do you link two HTML pages?
    • Using an <a href="page.html"> link.
  8. What does CSS stand for?
    • Cascading Style Sheets.
  9. Where is internal CSS written?
    • Inside a <style> tag within the <head>.
  10. What is an HTML comment?
    • Text between <!-- and --> not shown on the page.
  11. What does the table tag do?
    • It creates a table layout.
  12. What tag is used for an image?
    • <img> tag is used for images.
  13. What is a nested list?
    • A list inside another list.
  14. How do you style a table?
    • Using CSS for borders, spacing, and colors.

    • To include meta info, title, and links to styles.
